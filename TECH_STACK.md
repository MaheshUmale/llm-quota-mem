# Project Technology Stack

## 1. Environment & Core Engines
- Core Engine: Python 3.12+
- Primary Framework: FastAPI / Pydantic V2

## 2. Data Strategy
- Primary Database: SQLite (for local cache/memory)
- Vector Store: NumPy-based SimpleVectorStore
- Graph Engine: KnowledgeGraph (Triple-store)

## 3. Tooling & Linting
- Linter/Formatter: Ruff
- Testing Library: PyTest
